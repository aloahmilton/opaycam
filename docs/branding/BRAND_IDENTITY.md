# OpayCam Brand Identity

## Brand Overview
OpayCam is a comprehensive mobile finance platform for Cameroon, designed to empower users with seamless, secure, and accessible digital financial services.

## Color Palette

### Primary Colors
- **OpayCam Yellow**: `#FFD700` (Primary brand color)
  - Used for: Primary buttons, key highlights, active states
  - Conveys: Energy, optimism, trust, familiarity
  
- **Deep Corporate Blue**: `#003366`
  - Used for: Headers, navigation, text, backgrounds
  - Conveys: Trust, stability, professionalism, security

### Secondary Colors
- **Light Yellow**: `#FFF4CC` - Backgrounds, subtle highlights
- **Medium Blue**: `#0066CC` - Interactive elements, links
- **Success Green**: `#28A745` - Success messages, positive actions
- **Warning Orange**: `#FFA500` - Alerts, warnings
- **Error Red**: `#DC3545` - Errors, destructive actions
- **Neutral Gray**: `#F5F5F5` - Backgrounds, dividers

## Typography

### Font Family
- **Primary**: **Inter** (Modern, rounded sans-serif)
- **Fallback**: System fonts (SF Pro on iOS, Roboto on Android)

### Font Sizes
- **Heading 1**: 32px / Bold
- **Heading 2**: 24px / Bold
- **Heading 3**: 20px / SemiBold
- **Body Large**: 18px / Regular
- **Body**: 16px / Regular
- **Body Small**: 14px / Regular
- **Caption**: 12px / Regular

### Font Weights
- Regular: 400
- Medium: 500
- SemiBold: 600
- Bold: 700

## Iconography

### Style Guidelines
- **Style**: Minimalist, clear, rounded edges
- **Line Weight**: 2px stroke
- **Size**: 24x24px (standard), 32x32px (large), 16x16px (small)
- **Color**: Match context (primary color for active, gray for inactive)

### Icon Categories
- Financial actions (send, receive, save, loan)
- Utilities (electricity, water, TV, internet)
- Navigation (home, history, profile, settings)
- Status indicators (success, error, pending, info)

## UI Components

### Buttons
- **Primary**: Yellow background, white text, rounded (12px)
- **Secondary**: White background, blue border, blue text
- **Tertiary**: Transparent background, blue text

### Cards
- **Style**: Clean, minimal shadow (elevation 2)
- **Border Radius**: 16px
- **Padding**: 16px
- **Background**: White or light gray

### Input Fields
- **Border**: 1px solid light gray
- **Border Radius**: 8px
- **Padding**: 12px 16px
- **Focus State**: Blue border, subtle glow

## Spacing System
Based on 8px grid:
- **XXS**: 4px
- **XS**: 8px
- **SM**: 16px
- **MD**: 24px
- **LG**: 32px
- **XL**: 48px
- **XXL**: 64px

## Design Principles

1. **Clarity**: Every element should be immediately understandable
2. **Simplicity**: Minimal steps to complete any transaction
3. **Trust**: Security and reliability visible in every interaction
4. **Familiarity**: Intuitive navigation and recognizable patterns
5. **Accessibility**: Readable text, clear contrast, touch-friendly targets (min 44px)

## Voice & Tone
- **Friendly but Professional**: Approachable yet trustworthy
- **Clear and Direct**: No jargon, simple language
- **Supportive**: Helpful error messages and guidance
- **Confident**: Assured in our service and security
